A simple and intuitive web application to track your personal finances, manage expenses, and visualize your spending habits. Built with modern web technologies for responsive and interactive user experience.

---

## Features

- **Add Expenses:** Quickly log income and expenses with categories and descriptions.  
- **Expense History:** View a detailed history of all transactions.  
- **Visual Insights:** Charts and graphs to visualize spending trends over time.  
- **Responsive Design:** Works smoothly on desktop, tablet, and mobile devices.  
- **Light & Dark Mode:** Switch between themes for comfortable viewing.  

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Frameworks/Libraries:** React, Tailwind CSS, Framer Motion  
- **Data Storage:** Local Storage (client-side persistence)  
- **Charts:** Chart.js or equivalent (if used)
